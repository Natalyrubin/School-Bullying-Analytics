# 📊 School Bullying Analytics

## 🧠 Overview
A data-driven project focused on analyzing and understanding bullying patterns among students.  
The goal is to uncover behavioral signals, identify risk factors, and build predictive insights that can support early detection and intervention.

This project combines exploratory data analysis (EDA), data preprocessing, and machine learning techniques to extract meaningful insights from real-world data.

---

## 🎯 Objectives
- Analyze key factors associated with bullying behavior  
- Identify patterns across demographics, social behavior, and environment  
- Detect high-risk indicators using statistical and data analysis methods  
- Build a foundation for predictive modeling of bullying risk

## 🔍 Key Analysis Areas
- Bullying occurrence (on/off school property, cyber bullying)
- Social indicators (loneliness, friendships, support systems)
- Behavioral signals (absenteeism, physical conflicts)
- Demographic segmentation (age, gender)
